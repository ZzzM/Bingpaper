A practical iOS Bing wallpaper App 

## Features
- [x] Widgets
- [x] Switch to Dark & Light Modes Manually
- [x] App and widget localization (简体中文、English)
- [x] SwiftUI / Async / Await

## Compatibility
Requires **iOS 15.0** or later.
